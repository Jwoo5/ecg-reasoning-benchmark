# ecg-reasoning-benchmark
GEM만 현재 돌아가는 상태 (Pulse는 모델 로딩은 코드는 가져왔지만, 현재 pulse용 라바 버전이 아니여서 안돌아감)
GEM 인퍼런스로 현재 나오는 answer도 post-processing 해야되는 상태 (ex. ###Answer: (a) Yes 혹은 그냥 yes로 나옴)
checkpoints 디렉토리 만들어서 (https://github.com/YubaoZhao/ECG-Chat) 에서 ECG-CoCa 체크포인트 다운받아서 넣어줘야됨 (학회 억셉되서 어제부터 체크포인트 공개함)
